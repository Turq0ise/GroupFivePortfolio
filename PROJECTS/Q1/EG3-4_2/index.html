    • a. Comparison Operators

    (Equal to or ==); true if the value on the left is equal to the value on the right, otherwise it would be false.

    (Equal value and type or ===); true if the value is corresponding to the value on the right, otherwise it would be false.

    (!=); This operator is used to determine if a value is not equal.

    (!==); This operator is used if there is a â€˜not equal value or not equal type' comparison.

    (>); This sign means it's greater than the other number so 1>2 is true since 2 has a greater number/value than 1.

    (<); This sign means it's less than the other number so 1<2 is false since 2 has a greater number than 1 but 1<2 says that 1 has a greater number/value than 2 which is not a true statement.

    (>=); Greater than or equal to (>=) returns true if the value on the left is greater than or equal to the value on the right, otherwise it returns false.

    (<=); Less than or equal to (<=) returns true if the value on the left is less than or equal to the value on the right, otherwise it returns false.

    b. Logical Operators

    (&&); Is used to identify the variable if it's true or false, but if there's one statement as false, it will come out as false.

    (||); Is used to compare two variables and identify if one of them returns true.

    (!); Is used to return the opposite of the value..
