    What are Javascript Operators?

  • Arithmetic Operators
  • Logical Operators
  • Assignment Operators
  • References

    • a. Arithmetic Operators

    are used to perform mathematical expressions on numbers.

    (+); operator that performs Addition. To get the sum of two operands.

    (-); operator that performs subtraction. To get the difference of two operands.

    (*); operator that performs multiplication. To get the product of two operands.

    (/); operator that performs division. To get the quotient of two operands.

    Remainder or % is where you can find the remain number of what you divided, example is when you divide 9 by 2 or 9÷2 the remainder is 1 and if you type in the code 9%2=1 it will be equal to 1 in a simple explanation if you use this code it will find the remainder of the number that you want to divide.

    Exponentiation or ** is how many times you will times(x) the number itself, example is 5**2 you will times by by its own number by one 5x5=25 and if you do 5**3 times(x) you multiply it by itself 3 times 5x5x5=125, so on and so forth.

    Increment or ++ is add 1 or plus one to the number

    Decrement or -- is minus 1 to the number

    b. Logical Operators

    (&&); is used to identify the variable if its true or false but if there's one statement as false it will come out as false

    (||); is used to identify the variables but if one statement is true it will come out as true

    (!); Is used to switch the value of the two variables

    c. Assignment Operators

    (=); When using the “=” operator, the right operand's value is assigned to the left operand (variable).

    For example: Let's say the given is x = 5 and y = 10; If x = y, and x is 5 and y is 10, then x would be 10 since the value of the right operand (y), which is 10, is now the value of the left operand (x).

    (+=); When using the “+=” operator, this means that both left and right operands are added together, and the result should be the left operand's value.

    For example: Let's say that the assigned variable is x = 5 and the number to be added is 1. If x += 1, and x is equal to 5, then x would be 6.

    (-=); The subtraction assignment operator ( -= ) subtracts the value of the right operand from a variable and assigns the result to the variable.

    (*=); The multiplication assignment operator ( *= ) multiplies a variable by the value of the right operand and assigns the result to the variable.

    (/=); divides a variable to a number and the result of the variable will be the quotient.

    (%=); divides a variable to a number and the result of the variable will be the remainder.
